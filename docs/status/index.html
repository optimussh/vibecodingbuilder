OpenChamber 켜는 법

  1. pwsh scripts/fetch-openchamber.ps1
  2. cd vendor/openchamber; bun install --ignore-scripts; bun run build:ui; bun run build:web
  3. pwsh scripts/start-openchamber.ps1 또는 bun server/index.js --port 3001 (packages/web)
  4. .env: OPENCHAMBER_ENABLED=true · OPENCHAMBER_URL=http://127.0.0.1:3001
  5. npm run dev → 로그인 후 Chamber

검증 체크리스트

  • npm run db:up · npm run dev
  • 포털 :3000/ 열림
  • 레거시 :5173 user1 로그인 → RAG·채팅
  • /opencode/global/health 는 비로그인 시 401
  • /chamber 로그인 후 안내 또는 OC UI
  • user2가 user1 세션/문서 접근 불가
