# CodeHarbor 파일럿 Statement of Work (SoW) 템플릿

**문서 버전:** 2026-07-19  
**제품:** CodeHarbor — Governed Vibe Coding Platform  
**용도:** 고객 파일럿 계약·내부 견적 초안 (법률 검토 후 사용)

---

## 1. 당사자

| | |
|--|--|
| **공급자 (Provider)** | [회사명] |
| **고객 (Customer)** | [고객사명] |
| **파일럿 기간** | 계약 발효일로부터 **8주** (연장 협의 가능) |
| **파일럿 규모** | 최대 **[15]** named users |

---

## 2. 목적

고객 개발 조직이 **사내 정책(키 중앙화·격리·감사)** 하에서 바이브코딩(AI 에이전트 코딩)을  
안전하게 시험하고, 본계약(Team/Enterprise) 전환 여부를 결정한다.

---

## 3. 범위 (In Scope)

### 3.1 제공 소프트웨어 (Pilot tier)

| 항목 | 포함 |
|------|------|
| CodeHarbor Gateway | 인증, BFF, 테넌트 프록시, 포털 |
| 멀티유저 워크스페이스 격리 | 유저별 디렉터리 + 세션 소유권 |
| OpenCode 연동 에이전트 | 고객 제공 또는 공급자 대행 모델 키 |
| OpenChamber / 채팅 UI | IDE 셸 또는 레거시 채팅 |
| RAG 미니 (옵션) | 문서 업로드·검색·채팅 주입 |
| 감사 로그 | 관리자 조회·export (파일럿 종료 시 제공) |
| 사용량 요약 | 메시지/추정 비용 리포트 (가능한 범위) |
| 기술 지원 | 슬랙/메일 **영업일 기준 1 영업일 응답** |

### 3.2 공급자 수행 업무

1. 설치 지원: 고객 지정 환경(온프렘 VM 또는 고객 VPC) 또는 공급자 호스팅 PoC  
2. 관리자 교육 1회 (60–90분) + 개발자 온보딩 1회 (45분)  
3. 주 1회 체크인 (15–30분) × 파일럿 기간  
4. 종료 시: 성과 리포트 1부 (사용량, 주요 감사 지표, 권고)

### 3.3 고객 책임

1. 파일럿 참여자 명단·역할 지정 (Admin 1+, Developer N)  
2. IdP/네트워크/방화벽 협조 (SSO 사용 시)  
3. 모델 API 키 비용 (직접 계약 또는 공급자 패스스루 정산)  
4. 사내 보안 정책에 따른 데이터 분류·금지 데이터 고지  
5. 성공 기준 측정에 필요한 설문 협조 (종료 시)

---

## 4. 범위 외 (Out of Scope)

- 프로덕션 SLA / 24×7 지원  
- 고객 애플리케이션 기능 개발 대행  
- Air-gap 완전 폐쇄망 구축 (별도 Enterprise 견적)  
- 개인 Cursor/Copilot 라이선스 대체 보증  
- 모델 학습·파인튜닝  
- 본 문서에 명시되지 않은 커스텀 개발 (CR로 처리)

---

## 5. 성공 기준 (Exit Criteria)

아래 **3개 이상** 충족 시 파일럿 성공으로 간주하고 본계약 협의 개시.

| # | 지표 | 목표 (예시, 계약 시 수치 확정) |
|---|------|--------------------------------|
| 1 | 주간 활성 유저 / seat | ≥ 40% |
| 2 | 유저당 주간 에이전트 세션 | ≥ 3 |
| 3 | 크로스 테넌트 접근 사고 | **0** |
| 4 | 키 유출·브라우저 노출 | **0** |
| 5 | 종료 설문 “표준 도구로 검토할 의향” | ≥ 70% 긍정 |
| 6 | 보안/IT 파일럿 승인 유지 | 철회 없음 |

## 6. 일정 (표준 8주)

| 주 | 마일스톤 |
|----|----------|
| W0 | 킥오프, 환경 접근, 계정·키, 보안 1페이지 공유 |
| W1 | 설치 완료, Admin 교육, 스모크 테스트 |
| W2–W3 | 개발자 온보딩, 일상 사용, 이슈 수집 |
| W4 | 중간 리뷰 (사용량·피드백) |
| W5–W6 | 정책 조정 (쿼터, 프로젝트, SSO 등 합의 범위) |
| W7 | 성과 데이터 freeze, 설문 |
| W8 | 종료 리포트, 본계약/종료 결정 |

## 9. 지적재산 · 비밀유지

- 공급자 소프트웨어·문서 IP는 공급자 보유; 파일럿 기간 **비독점·비양도 사용권**  
- 고객 코드·데이터 IP는 고객 보유  
- 양측 NDA 또는 본 SoW 비밀유지 조항 적용  
- 파일럿 중 생성된 익명 사용 통계는 제품 개선에 사용 가능 (개인 식별 제외)

---

## 10. 종료 · 데이터 반환

1. 기간 만료 또는 양측 합의 종료  
2. 고객 요청 시 워크스페이스 아카이브 지원 (합의 형식)  
3. 공급자 호스팅 PoC인 경우: 종료 후 **[14]일** 내 데이터 삭제 확인서  
4. 본계약 미체결 시 라이선스 종료
