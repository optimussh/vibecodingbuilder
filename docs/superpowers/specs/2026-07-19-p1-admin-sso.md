## 4. Architecture

```
Browser
  │  cookie session
  ▼
CodeHarbor Gateway
  ├─ /login (local) ────────────────► users table / env passwords
  ├─ /api/auth/oidc/* ──────────────► IdP (authorize + code + token)
  ├─ /api/admin/* (role=admin) ─────► users, credentials, quotas
  └─ runtime ───────────────────────► reads credentials from vault (memory+DB)
Postgres
  ├─ web_sessions (existing)
  ├─ users (new or extend)
  ├─ auth_identities (oidc sub ↔ user)
  ├─ api_credentials (encrypted blob)
  └─ audit_events (existing path)
```

### Credential vault (BFF invariant)

- 키는 **서버 프로세스 + DB ciphertext** 에만 존재  
- `GET /api/admin/credentials` 는 **메타만** (provider, last4, updatedAt) — plaintext 재조회 없음  
- `PUT` 시 plaintext body → encrypt → store → process.env / in-memory cache 갱신  
- 암호화: `CREDENTIALS_MASTER_KEY` (32-byte base64) AES-256-GCM

## 5. Data model

```sql
-- users: local + oidc linked
CREATE TABLE IF NOT EXISTS app_users (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  username      TEXT UNIQUE NOT NULL,
  display_name  TEXT,
  role          TEXT NOT NULL DEFAULT 'user' CHECK (role IN ('admin','user')),
  password_hash TEXT,                  -- null if oidc-only
  disabled      BOOLEAN NOT NULL DEFAULT false,
  daily_quota   INT,                   -- null = org default
  created_at    TIMESTAMPTZ NOT NULL DEFAULT now(),
  updated_at    TIMESTAMPTZ NOT NULL DEFAULT now()
);

CREATE TABLE IF NOT EXISTS auth_identities (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id       UUID NOT NULL REFERENCES app_users(id) ON DELETE CASCADE,
  provider      TEXT NOT NULL,          -- 'oidc' | 'local'
  subject       TEXT NOT NULL,         -- oidc `sub`
  email         TEXT,
  raw_claims    JSONB,
  UNIQUE (provider, subject)
);

CREATE TABLE IF NOT EXISTS api_credentials (
  provider      TEXT PRIMARY KEY,      -- 'google' | 'openai' | ...
  ciphertext    BYTEA NOT NULL,
  iv            BYTEA NOT NULL,
  tag           BYTEA NOT NULL,
  last4         TEXT NOT NULL,
  updated_by    TEXT,
  updated_at    TIMESTAMPTZ NOT NULL DEFAULT now()
);
```

### Role mapping (OIDC)

```env
OIDC_ISSUER=https://login.microsoftonline.com/{tenant}/v2.0
OIDC_CLIENT_ID=...
OIDC_CLIENT_SECRET=...
OIDC_REDIRECT_URI=http://127.0.0.1:3000/api/auth/oidc/callback
OIDC_ADMIN_GROUP=CodeHarbor-Admins   # optional; claim path configurable
OIDC_GROUPS_CLAIM=groups
```

규칙:

1. 로그인 성공 → `auth_identities` upsert → `app_users`  
2. `groups` 에 admin 그룹 있으면 `role=admin`, 아니면 `user` (기존 유저 admin 유지 옵션)  
3. `disabled=true` 이면 세션 거부

## 6. API

### Auth

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/api/auth/oidc/login` | public | 302 → IdP |
| GET | `/api/auth/oidc/callback` | public | code exchange → session |
| GET | `/api/auth/me` | user | user + role + auth methods |
| POST | `/api/auth/login` | public | local (existing) |
| POST | `/api/auth/logout` | user | existing |

### Admin (require `role=admin`)

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/admin/users` | list users |
| PATCH | `/api/admin/users/:id` | role, disabled, daily_quota |
| POST | `/api/admin/users` | create local user |
| GET | `/api/admin/credentials` | list providers + last4 (no secrets) |
| PUT | `/api/admin/credentials/:provider` | body `{ apiKey: string }` |
| DELETE | `/api/admin/credentials/:provider` | remove key |
| GET | `/api/admin/settings` | oidc enabled?, local login allowed? |

## 8. Sequence — OIDC login

```
User → GET /api/auth/oidc/login
Gateway → 302 IdP (state, nonce in session)
User → IdP consent
IdP → GET /api/auth/oidc/callback?code&state
Gateway → token endpoint
Gateway → userinfo / id_token claims
Gateway → upsert identity + user
Gateway → req.session.user = { id, username, role }
Gateway → 302 next=/
```
